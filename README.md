# PoliMirror（政治の鏡）

> 深淵をのぞく時、深淵もまたこちらをのぞいているのだ — ニーチェ

**ミッション**: 政治家の言葉を、すべて記録する。それだけで、政治は変わる。

## 大原則
- 逃げは許さない。そのかわり、立派なことも全て記録する。
- 一次ソースのみ記録する
- 事実と分析を明確に分離する
- 収集方法・判定方法を全て公開する

## 対象
- 国会議員（衆465・参248）
- 知事・市区町村長・地方議会議員
- 裁判官・政治評論家
- 合計約35,000人

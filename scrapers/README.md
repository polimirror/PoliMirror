# PoliMirror スクレイパー

## 収集ポリシー
詳細は docs/collection_policy.md を参照。

## 基本ルール
- robots.txtを必ず遵守する
- リクエスト間隔: 最低5秒
- 実行時間帯: 深夜0時〜6時のみ
- ログインが必要なページは収集しない
- User-Agentに連絡先を明記する

## ディレクトリ構成
- `diet/` - 国会議事録スクレイパー
- `finances/` - 政治資金収支報告書スクレイパー
- `elections/` - 選挙公報スクレイパー
- `official_sites/` - 議員公式サイトスクレイパー
